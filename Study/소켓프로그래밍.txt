== 서버 

소켓도 파일로 간수 read And write를 이용한 데이터를 송수신 그러나 윈도우는 구분이 되어있어 소켓기반 입출력
함수가 존재

int send(SOCKET s, const char*, buf, int len, int flags);
	= 성공 시 전송된 바이트 수 실패시 SOCKET_ERROR 반환, 
int recv(SOCKET s, const char*, buf, len len, int flags);
	= 성공 시 수신한 바이트수 (EOF->0), 실패시 SOCKET_ERROR 반환

소켓의 타입이란 소켓의 데이터 전송방식을 의미, 이 정보를 socket 함수의 두 번째 인자로 전달,
첫 번째 인자로 프로토콜 체계 정보를 입력하긴 하지만 한 프로토콜 체계에서도 둘 이상의 데이터 전송방식이 존재하기
때문에 따로 인자를 넣어줘야 한다.
